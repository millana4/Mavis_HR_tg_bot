# Mavis HR Telegram Bot
Telegram-бот для HR-отдела компании МАВИС. Предоставляет сотрудникам доступ к справочной информации, контактам, документам и пульс-опросам.

## Функциональность
- Ролевая модель: новички (до 3 месяцев) и действующие сотрудники
- Динамическое меню: контент загружается из таблиц NocoDB
- Поиск сотрудников: по ФИО, отделу, локации с поддержкой нечеткого поиска
- Пульс-опросы: автоматические (1 неделя, 1, 3, 6 месяцев, год) и ручные (при увольнении)
- Формы обратной связи: интерактивные опросы с сохранением ответов
- Рассылки: для администраторов с возможностью отложенной отправки
- Авторизация: по номеру телефона

## Технологии
- Python 3.12
- aiogram 3.x
- NocoDB (база данных)
- Асинхронные запросы (aiohttp)

## Запуск
1. Клонируйте репозиторий
2. Настройте виртуальное окружение
3. Установите зависимости: pip install -r requirements.txt
4. Настройте переменные окружения. Скопируйте файл .env.example в .env и установите свои значения
5. Запустить бота python main.py
